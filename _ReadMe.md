### 运行说明

- dist内为工程文件，直接在dist目录下运行http-server即可
- npm install -g http-server
- http-server -p 1235

项目介绍

-  convert_obj_to_json 是用于转换obj为json文件
- 项目整体用parcel 打包
- npm install -g parcel
- parcel index.html -p 1234 即可
